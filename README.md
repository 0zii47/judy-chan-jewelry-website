# Judy Chan Jewelry Website

Static five-page website for Judy Chan and Chan Fine Jewelry.

## Pages

- `index.html` - home
- `about.html` - designer story
- `collection.html` - filterable jewelry collection
- `bespoke.html` - bespoke process
- `contact.html` - appointment contact page

## Preview

Open `index.html` directly in a browser, or run a local static server from this folder:

```bash
python3 -m http.server 4173
```

Then browse `http://127.0.0.1:4173/`.

## GitHub Pages

Publish the repository from the `main` branch root in GitHub Pages settings.
The site entry point is `index.html`, so no build step is required.

## Assets

- Final WebP website images are in `assets/images/`.
- Page screenshots are in `assets/screenshots/`.
- The portrait and sketch assets were selected from the supplied Judy source archives.
- Collection and studio images were completed with project-specific generated jewelry photography where the source archive did not provide a consistent website-ready set.
